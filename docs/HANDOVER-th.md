# คู่มือดูแลเว็บไซต์ร้าน (สำหรับครอบครัว ใช้จากมือถือได้เลยค่ะ)

ทุกอย่างในคู่มือนี้ทำจากโทรศัพท์ได้ค่ะ เข้าไปที่ **https://apple-vegan-cafe.com/admin**
หน้าจอระบบเป็นภาษาอังกฤษ แต่ไม่ต้องกังวลนะคะ — คู่มือนี้บอกทุกปุ่มที่ต้องกดทีละขั้น
สิ่งสำคัญที่สุดที่ต้องจำ: แก้อะไรเสร็จให้กดปุ่ม **"Save"** แล้วรอประมาณ **3 นาที** การเปลี่ยนแปลงจะขึ้นเว็บจริงเองค่ะ

## 1. การเข้าสู่ระบบครั้งแรก

บ้านเรามีบัญชี GitHub **บัญชีเดียวใช้ร่วมกันทั้งครอบครัว** ค่ะ (ชื่อผู้ใช้และรหัสผ่านถามคนในครอบครัวที่ดูแลได้เลย)

1. เปิด Safari (iPhone) หรือ Chrome (Android) แล้วพิมพ์ `apple-vegan-cafe.com/admin`
2. แตะปุ่ม **"Sign In with GitHub"**
3. ใส่ชื่อผู้ใช้และรหัสผ่านของบัญชี GitHub ครอบครัว แล้วแตะ **"Sign in"**
4. ครั้งแรก GitHub อาจส่งรหัสยืนยันไปที่อีเมลของครอบครัว — เปิดอีเมล จดรหัส แล้วนำมากรอกค่ะ
5. ถ้ามีหน้าจอถามสิทธิ์ ให้แตะปุ่มสีเขียว **"Authorize"** (ถามแค่ครั้งแรกครั้งเดียวค่ะ)
6. เข้าสำเร็จแล้วจะเห็นเมนูรายการ เช่น **"Menu dishes"**, **"Blog (ไทย)"**, **"Site settings"**

**ทำให้เปิดง่ายเหมือนแอป (แนะนำมากค่ะ):**

- iPhone: ในหน้า /admin แตะปุ่มแชร์ (สี่เหลี่ยมมีลูกศรขึ้น) → เลื่อนหา **"Add to Home Screen"** → **"Add"**
- Android: แตะจุดสามจุดมุมขวาบน → **"Add to Home screen"** → **"Add"**
- ต่อไปแค่แตะไอคอนบนหน้าจอโฮมก็เข้าได้เลย ไม่ต้องพิมพ์อะไรอีกค่ะ

> 📸 [ภาพหน้าจอจะเพิ่มภายหลัง: หน้าเข้าสู่ระบบที่มีปุ่ม "Sign In with GitHub" และเมนูรายการหลังเข้าสำเร็จ]

## 2. เปลี่ยนราคาอาหาร

1. แตะ **"Menu dishes"** — จะเห็นรายการอาหารทั้งหมด พร้อมราคา (฿) ต่อท้ายชื่อ
2. หาจานที่ต้องการ (พิมพ์ชื่อในช่องค้นหาด้านบนได้ค่ะ) แล้วแตะเข้าไป
3. เลื่อนหาช่อง **"Price (THB)"** ลบราคาเดิม แล้วพิมพ์ราคาใหม่เป็นตัวเลขอย่างเดียว เช่น `159` (ไม่ต้องใส่ ฿ หรือคำว่าบาทนะคะ)
4. (ไม่บังคับ) แตะช่อง **"Reviewed on"** เลือกวันนี้ เพื่อจดไว้ว่าตรวจราคาแล้วค่ะ
5. แตะปุ่ม **"Save"** ด้านบน
6. รอ 3 นาที แล้วเปิดหน้าเว็บร้านดูราคาใหม่ได้เลยค่ะ

> 📸 [ภาพหน้าจอจะเพิ่มภายหลัง: หน้าแก้ไขจานอาหาร ชี้ตำแหน่งช่อง "Price (THB)" และปุ่ม "Save"]

## 3. ซ่อนเมนูที่หมด (สต๊อปลิสต์) — และวิธีเอากลับมา

ของหมดวันนี้? ไม่ต้องลบเมนูนะคะ แค่ปิดการแสดงชั่วคราวพอค่ะ

1. แตะ **"Menu dishes"** แล้วแตะจานที่หมด
2. หาช่องติ๊ก **"Available (uncheck = stop list)"** แล้ว**เอาเครื่องหมายถูกออก**
3. แตะ **"Save"** — อีก 3 นาทีจานนี้จะหายไปจากเมนูบนเว็บค่ะ

**พอของกลับมามีแล้ว:** เข้าจานเดิม **ติ๊กถูกกลับเข้าไป**ที่ช่อง **"Available (uncheck = stop list)"** แล้วแตะ **"Save"** — จานจะกลับมาโชว์เหมือนเดิมค่ะ

เกร็ดเล็กๆ: ช่อง **"Show on home page"** คือการเลือกให้จานนั้นโชว์เด่นบนหน้าแรกของเว็บค่ะ

> 📸 [ภาพหน้าจอจะเพิ่มภายหลัง: ช่องติ๊ก "Available (uncheck = stop list)" ตอนติ๊กอยู่ และตอนเอาติ๊กออก]

## 4. เพิ่ม/เปลี่ยนรูปอาหาร

1. แตะ **"Menu dishes"** แล้วแตะจานที่ต้องการ
2. เลื่อนหาหัวข้อ **"Photos"**
3. แตะปุ่มเพิ่มรูป **"Add Image"** ใต้หัวข้อนั้น
4. เลือก **"Upload"** แล้วเลือกรูปจากคลังภาพในมือถือได้เลยค่ะ (ถ่ายใหม่ตอนนั้นก็ได้)
5. อยากลบรูปเก่า ให้แตะที่รูปนั้นแล้วเลือกลบ (**"Remove"**) ค่ะ
6. แตะ **"Save"**

เคล็ดลับรูปสวย: ถ่ายตอนกลางวันใกล้หน้าต่าง ถือกล้องนิ่งๆ ให้เห็นจานเต็มๆ ค่ะ

> 📸 [ภาพหน้าจอจะเพิ่มภายหลัง: หัวข้อ "Photos" กับปุ่ม "Add Image" และหน้าต่างเลือกรูปจากมือถือ]

## 5. เปลี่ยนเวลาเปิด-ปิด และวันหยุดพิเศษ

1. แตะ **"Site settings"** แล้วแตะ **"Address & opening hours"**
2. ในหัวข้อ **"Pattaya"** เลื่อนหา **"Opening hours"** — แต่ละแถวมี:
   - **"Days"** = วันที่ใช้เวลานี้ (mon=จันทร์, tue=อังคาร, wed=พุธ, thu=พฤหัส, fri=ศุกร์, sat=เสาร์, sun=อาทิตย์)
   - **"Opens (HH:MM)"** = เวลาเปิด และ **"Closes (HH:MM)"** = เวลาปิด — ใช้เวลาแบบ 24 ชั่วโมง เช่น `09:00` และ `20:30` ค่ะ
3. แก้เสร็จแตะ **"Save"**

**วันหยุดพิเศษ / เทศกาล:** ในหน้าเดียวกัน เลื่อนหา **"Special days (holidays)"**

1. แตะปุ่มเพิ่มรายการใหม่
2. **"Date (YYYY-MM-DD)"** = ใส่วันที่แบบ ปี-เดือน-วัน เช่นวันสิ้นปี ใส่ `2026-12-31`
3. ถ้า**ปิดทั้งวัน** ให้เว้นช่อง **"Opens (empty = closed)"** ว่างไว้เลยค่ะ / ถ้าเปิดสั้นลง ใส่เวลาเปิด-ปิดของวันนั้น
4. **"Note"** = ข้อความบอกลูกค้า เช่น `หยุดวันสงกรานต์`
5. แตะ **"Save"** — เว็บจะแจ้งลูกค้าให้เองค่ะ

> 📸 [ภาพหน้าจอจะเพิ่มภายหลัง: หน้า "Address & opening hours" ส่วน "Opening hours" และ "Special days (holidays)"]

## 6. เขียนบทความ (Blog ไทย)

1. แตะ **"Blog (ไทย)"** แล้วแตะปุ่มสร้างบทความใหม่ (**"Create"** / เครื่องหมาย +)
2. กรอกช่องต่างๆ ค่ะ:
   - **"Title"** = ชื่อบทความ (ภาษาไทยได้เลย)
   - **"Short description (meta)"** = คำโปรยสั้นๆ 1–2 ประโยค
   - **"URL slug"** = ชื่อลิงก์ เป็นภาษาอังกฤษตัวเล็ก คั่นด้วยขีด เช่น `songkran-menu-2026`
   - **"Translation key (same across languages)"** = ใส่ค่าเดียวกับ URL slug ได้เลยค่ะ
   - **"Published"** = วันที่ลงบทความ
   - **"Text"** = เนื้อหาบทความ พิมพ์ได้ยาวเท่าที่ต้องการ ใส่รูปได้ด้วยค่ะ
3. ช่องติ๊ก **"Draft"** สำคัญมากค่ะ: **ติ๊ก "Draft" อยู่ = บทความยังไม่แสดงบนเว็บ** (เก็บเป็นฉบับร่าง เขียนต่อวันหลังได้)
4. เขียนเสร็จ พร้อมเผยแพร่แล้ว ให้**เอาติ๊ก "Draft" ออก** แล้วแตะ **"Save"** — อีก 3 นาทีบทความขึ้นเว็บค่ะ

> 📸 [ภาพหน้าจอจะเพิ่มภายหลัง: หน้าเขียนบทความใหม่ ชี้ช่อง "Title", "Text" และช่องติ๊ก "Draft"]

## 7. สิ่งที่ไม่ควรแก้เอง (ให้ติดต่อผู้พัฒนาค่ะ)

ช่องเหล่านี้เกี่ยวกับลิงก์และระบบหลังบ้าน ถ้าแก้ผิด ลิงก์เก่าที่ลูกค้าเซฟไว้หรือที่ Google จำได้จะพังค่ะ:

- **"URL slug (per language)"** ในเมนูอาหาร และ **"URL slug"** ของบทความที่เผยแพร่ไปแล้ว
- **"Previous slugs (for redirects)"**
- **"Translation key (same across languages)"** ของบทความที่มีหลายภาษาแล้ว
- **"Data source"** ในเมนูอาหาร

อยากเปลี่ยนชื่อลิงก์หรือย้ายบทความ ไม่ต้องลองเองนะคะ ทักผู้พัฒนาได้เลย เดี๋ยวจัดการให้ค่ะ

## 8. เปิด-ปิดรับออเดอร์ Grab และแก้เวลาใน Google Maps

เว็บไซต์ของเราแสดง "เวลาหน้าร้าน" (7:30–22:30) เท่านั้นค่ะ ส่วนการรับออเดอร์กลางคืนหรือการพักรับออเดอร์ชั่วคราว
ทำในแอปของแต่ละที่โดยตรง:

**พักรับออเดอร์ Grab ชั่วคราว (เช่น อยากพักผ่อนคืนนี้):**

1. เปิดแอป **GrabMerchant**
2. ที่หน้าหลักร้าน มองหาปุ่มสถานะร้าน แล้วเลือก **"Pause new orders"** (พักรับออเดอร์)
3. เลือกระยะเวลา แล้วยืนยัน — พอครบเวลาระบบเปิดรับเองค่ะ (หรือกดเปิดเองก่อนก็ได้)
4. ลูกค้าจะเห็นสถานะจริงในแอป Grab เสมอ ไม่ต้องแก้อะไรบนเว็บไซต์นะคะ

**แก้เวลาเปิด-ปิดใน Google Maps:**

1. เปิดแอป **Google Maps** แล้วล็อกอินบัญชีที่เป็นเจ้าของร้าน
2. ค้นหาร้านเรา → แตะโปรไฟล์ร้าน → **"Edit profile"** → **"Hours"**
3. แก้เวลา หรือเพิ่ม **"Special hours"** สำหรับวันหยุดพิเศษ แล้วบันทึก

จำง่ายๆ ค่ะ: **เว็บไซต์ = แก้ใน /admin · Grab = แอป GrabMerchant · Google = แอป Google Maps**
ถ้าเวลาหน้าร้านเปลี่ยนถาวร (เช่น เปิด 8:00 แทน) ต้องแก้ทั้งสามที่ให้ตรงกันนะคะ

## 9. ถ้ามีปัญหา

1. **แก้แล้วเว็บยังไม่เปลี่ยน?** ใจเย็นๆ ก่อนค่ะ รอให้ครบ 3 นาที แล้วรีเฟรชหน้าเว็บ (ปัดหน้าจอลง)
2. ยังไม่ขึ้นอีก ลองปิดเบราว์เซอร์แล้วเปิดหน้าเว็บใหม่ค่ะ
3. **เข้า /admin ไม่ได้ หรือหน้าจอขึ้น error?** ลองแตะ **"Sign In with GitHub"** เข้าสู่ระบบใหม่อีกครั้ง
4. ถ้ายังไม่ได้ ไม่ต้องพยายามแก้เองนะคะ — ส่งข้อความหาผู้พัฒนาได้เลย พร้อมแนบภาพหน้าจอ
