# คู่มือสำหรับครอบครัว (Owner's guide — Thai)

> สถานะ: โครงร่าง — จะเขียนให้เสร็จพร้อมภาพหน้าจอก่อนส่งมอบ (P1 backlog item 12).

## สิ่งที่ทำได้เองจากมือถือ ผ่าน apple-vegan-cafe.com/admin

1. **เปลี่ยนราคาอาหาร** — Menu dishes → เลือกจาน → Price (THB) → Save
2. **ซ่อนเมนูที่หมด (สต๊อปลิสต์)** — เลือกจาน → ปิดสวิตช์ "Available" → Save
3. **เปลี่ยนเวลาเปิด-ปิด** — Site settings → Address & opening hours
4. **เพิ่มรูปอาหาร** — เลือกจาน → Photos → อัปโหลด
5. **เขียนบทความ** — Blog (ไทย) → New

การเปลี่ยนแปลงจะขึ้นเว็บไซต์ภายใน ~3 นาที
